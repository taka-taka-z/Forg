# Forg — 組織摩擦係数診断ツール

> 「なぜゼロトラスト導入はいつも途中で止まるのか」  
> その答えは技術ではなく、組織の中にある。

## これは何か

Forg（フォルグ）は、企業のゼロトラスト推進（インフラの全体最適化）を阻む**組織摩擦**を
定量的に可視化するブラウザベースの診断ツールです。

独自指標「組織摩擦係数 F_org」をベースに、
現状のボトルネックと優先アクションを即座に導出します。

## 主な機能

- **F_org スコア算出** — 戦略・現場・変革の3軸から摩擦係数を計算
- **矛盾検出ロジック** — 回答間の整合性を自動チェック
- **アーキタイプ診断** — 組織を4象限マトリクスで分類
- **推定インパクト算出** — 摩擦が生む機会損失を可視化
- **優先アクション提言** — スコアに基づく具体的な次の一手を提示

## 使い方

ローカルに `forg.html`を保存し、ブラウザで開くだけ。

## 対象ユーザー

ゼロトラスト導入を検討・推進中のCISO・セキュリティアーキテクト・
DX推進担当者。Architecture Workshopの AS-IS セッションでの
ファシリテーションツールとしても活用可能。
